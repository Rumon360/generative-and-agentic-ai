## Feed Forward, Add & Norm, Linear, and Softmax in Transformer

After **Multi-Head Attention**, the Transformer applies a few more important steps to refine and produce the final output.

---

## 1. Feed Forward Network (FFN)

Once attention is applied, each token embedding is passed through a **Feed Forward Neural Network**.

### What does it do?

- It processes each token **independently**
- Applies non-linear transformations
- Helps the model learn more complex patterns

### How it works:

For each token vector:

1. Pass through a linear layer
2. Apply an activation function (like ReLU)
3. Pass through another linear layer

👉 In simple terms:

- It **transforms** the embedding into a richer representation

---

### Intuition

Think of it like:

> “Now that I understand the context (attention), let me think deeper about each word individually.”

---

## 2. Add & Norm (Residual Connection + Layer Normalization)

After both **Multi-Head Attention** and **Feed Forward**, we apply something called **Add & Norm**.

---

### Step 1: Add (Residual Connection)

We add the original input back to the output:

```
Output = Input + Processed_Output
```

### Why?

- Prevents loss of important information
- Helps with training very deep networks
- Keeps gradients stable

---

### Step 2: Layer Normalization

After adding, we normalize the values:

- Keeps numbers in a stable range
- Speeds up training
- Improves performance

---

### Intuition

> “Don’t forget the original meaning, just refine it.”

---

## 3. Linear Layer

After passing through several Transformer blocks, we apply a **Linear Layer**.

### What does it do?

- Converts the final embeddings into a vector of size equal to the vocabulary
- Each value represents a score for a possible next word

---

### Example

If vocabulary size = 10,000:

```
Embedding → [0.2, -0.5, ...] → Linear → [score1, score2, ..., score10000]
```

Each score corresponds to a word.

---

## 4. Softmax

Finally, we apply the **Softmax function**.

### What does it do?

- Converts scores into probabilities
- Ensures all values are between 0 and 1
- Total probability sums to 1

---

### Example

Before Softmax:

```
[2.1, 1.3, 0.2]
```

After Softmax:

```
[0.65, 0.25, 0.10]
```

---

### Intuition

> “Which word is most likely to come next?”

---

## Final Flow Summary

1. Multi-Head Attention → Understand context
2. Add & Norm → Stabilize and retain information
3. Feed Forward → Deep processing
4. Add & Norm → Again refine
5. Linear → Map to vocabulary
6. Softmax → Convert to probabilities
